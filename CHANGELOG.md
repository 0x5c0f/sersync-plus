## [0.1.0] - 2026-01-21

### 新增
- **核心同步功能**
  - 实时文件监控（inotify/watchdog）
  - Rsync 集成和多目标同步
  - 事件智能合并和过滤
  - 失败重试机制

- **Web 管理界面**
  - 实时监控仪表盘
  - 同步历史记录和统计
  - REST API 接口
  - WebSocket 实时推送
  - Basic Auth 认证

- **数据库存储**
  - SQLite 数据库集成
  - 同步历史永久存储
  - 分页查询和性能优化
  - 自动数据清理

- **通知系统**
  - Apprise 集成（支持 100+ 服务）
  - 灵活的通知规则引擎
  - 自定义消息模板
  - 批量通知和定时报告

- **双向同步**
  - 基于元数据的双向同步
  - 冲突检测和解决策略
  - 安全的元数据存储
  - 多节点支持

- **配置增强**
  - 完全兼容原版 sersync XML 配置
  - 命令行参数扩展
  - 数据库和日志路径配置
  - 环境变量支持

### 修复
- Python 3.12 子进程兼容性问题
- 目录删除时的 rsync 警告
- 配置解析错误处理
- 事件处理竞争条件

### 性能优化
- 同步历史记录分页加载
- 数据库索引优化
- 内存缓存机制
- 虚拟滚动支持

### 安全
- 元数据文件安全存储
- 路径遍历攻击防护
- 输入验证和清理
- 访问控制和认证

## 版本说明

### 版本号格式
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型
- **新增**：新功能
- **变更**：对现有功能的变更
- **弃用**：不再建议使用的功能
- **移除**：已删除的功能
- **修复**：问题修复
- **安全**：安全相关的修复
